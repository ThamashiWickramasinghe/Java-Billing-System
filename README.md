The **Cafe Mocha Billing System** is a Java-based restaurant billing application designed to manage customer details, menu items, orders, and bill calculations efficiently. The system helps streamline order processing and generate accurate bills with tax and discount support.

## Features
- Secure login system
- Store, update, view, and delete customer information
- Manage item menu with prices and descriptions
- Enter and store customer orders
- Display previous orders using order ID
- Calculate bills with tax and discount options
- Simple console-based user interface
