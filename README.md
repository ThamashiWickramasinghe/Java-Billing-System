The **Cafe Mocha Billing System** is a Java-based restaurant billing application designed to manage customer details, menu items, orders, and bill calculations efficiently. This system helps reduce manual work and ensures accurate billing with support for taxes and discounts.

## Features
- Store, update, view, and delete customer information
- Manage item menu with prices and descriptions
- Enter and store customer orders
- Display previous orders using order ID
- Calculate bills with tax and discount options
- Simple and user-friendly console-based interface

## Technologies Used
- Java
- File Handling
- Object-Oriented Programming (OOP) concepts

## System Users
- **Admin**: Manages customers, menu items, orders, and billing
